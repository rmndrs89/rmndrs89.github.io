Predicting the Eredivisie using a Bayesian Approach

Robbin Romijnders

University Hospital Schleswig-Holstein

26 May 2026

Introduction

  • Predicting match outcomes

About

  • Eredivisie consists of \(T = 18\) teams
    • Each team play twice against each other team
  • There are \(G = 306\) games

Model

  • The goals scored in the \(g\)-th game by the home and away team are: \[y_{g,1}, \quad\text{and}\quad y_{g,2}\]

  • These are counts, and can take values \(0, 1, \ldots\), typically modelled as following a Poisson distribution \[y_{g,j} \mid \theta_{g,j} \sim \mathrm{Poisson}(\theta_{g,j})\] with the parameters \(\theta_{g,j}\) representing the scoring intensity in the \(g\)-th game for the team playing at home (\(j=1\)) and away (\(j=2\)), respectively.

  • The parameters assume a log-linear random effect model: \[\begin{align} \log\left(\theta_{g,1}\right) &= \eta + \alpha_{h[g]} + \delta_{a[g]} \\ \log\left(\theta_{g,2}\right) &= \alpha_{a[g]} + \delta_{h[g]} \end{align}\]

Priors

We assume the following priors: \[ \begin{align} \eta &\sim \mathrm{Normal}(0, 0.0001) \\ \alpha_{t} &\sim \mathrm{Normal}(\mu_{\alpha}, \tau_{\alpha}) \\ \delta_{t} &\sim \mathrm{Normal}(\mu_{\delta}, \tau_{\delta}) \end{align} \]

We specify identifiability constrains, i.e., sum-to-zero constraint: \[\sum_{t=1}^{T}\alpha_{t} = 0, \quad \sum_{t=1}^{T}\delta_{t} = 0\]

We define the following hyperpriors: \[ \begin{align} \mu_{\alpha} &\sim \mathrm{Normal}(0, 0.0001), &\quad \mu_{\delta} &\sim \mathrm{Normal}(0, 0.0001) \\ \tau_{\alpha} &\sim \mathrm{Gamma}(0.1, 0.1), &\quad \tau_{\delta} &\sim \mathrm{Gamma}(0.1, 0.1) \end{align} \]
